# Louisiana Vehicle Protection Notes

This repository contains practical reference notes for vehicle owners in Louisiana, focusing on heat, UV exposure, oxidation, and common causes of exterior wear.

Topics include:
- How window tint reduces heat and UV exposure
- Ceramic coating vs wax vs sealants
- Paint correction basics (swirls, haze, oxidation)
- Preventing rock chips and physical damage (PPF considerations)
- Simple maintenance habits that help preserve a finish long-term

A complete local handbook is available here:

[https://sites.google.com/view/louisiana-vehicle-protection-handbook/](https://sites.google.com/view/louisiana-vehicle-protection)

These notes are intended as general guidance for vehicle owners researching protection options and maintenance best practices.
